# 금시세 일일 업데이트 서비스 — PRD

## 1. 개요

### 1.1 배경
국내 실물 금(3.75g·1돈 기준) 고시가는 거래소·은행·귀금속 업계에서 매일 여러 차례 갱신됩니다. 사용자는 **최신 고시일·살 때·팔 때(순금·18K·14K)** 를 한 화면에서 빠르게 확인하고, 과거 추이를 참고할 수 있기를 기대합니다.

### 1.2 목표
- **매일 자동으로** 공개 금시세 데이터를 수집·저장하고, 웹에서 테이블·(선택) 차트로 제공한다.
- 데이터 출처·갱신 시각·면책 문구를 명시하여 신뢰도와 투명성을 확보한다.

### 1.3 범위 (MVP)
- 금(Au) 3.75g 기준: 고시일, 내가 살 때 순금, 내가 팔 때 순금·18K·14K.
- 일별 히스토리 조회(페이지네이션), 최근 N건 엑셀 내보내기.
- (후속) 은·백금 등 확장, 사용자 알림, 관리자 대시보드.

## 3. 기능 요구사항

### 3.1 데이터 수집
- **소스**: 한국금거래소 공개 금시세 API (`POST /api/price/chart/list`, 본 프로젝트 POC에서 사용).
- **주기**: 운영 환경에서는 **매 영업일 1회 이상**(또는 사이트 고시 주기에 맞춤) 크론/스케줄러 실행.
- **중복 처리**: 동일 `date`(고시 시각 포함) 또는 비즈니스 키(고시일+차수) 기준 upsert.
- **보관**: 최소 1년 이상(정책에 따라 조정).

### 3.2 화면
- 메인: Tabulator 스타일 표 — 컬럼 `고시날짜`, `내가 살 때(3.75g) 순금`, `내가 팔 때(3.75g) 순금·18K·14K`.
- 필터: 기간 프리셋(예: 1주·3개월·전체), 사용자 지정 시작·종료일.
- 정렬: 기본 최신순.
- (선택) 차트: 살 때·팔 때 추이 — 소스 사이트와 유사한 UX.

### 3.3 내보내기
- **엑셀(xlsx)**: 현재 필터 결과 또는 상위 N건(예: 100건) 다운로드.
- 인코딩·숫자 서식: 천 단위 콤마, 날짜 `YYYY.MM.DD` 또는 ISO.

### 3.4 비기능
- **성능**: 목록 API 응답 2초 이내(캐시·페이지 크기 제한).
- **가용성**: 수집 실패 시 재시도·로그, 사용자에게 “마지막 성공 시각” 표시.
- **법·컴플라이언스**: 재배포 시 출처 표기, robots/API 이용약관 준수, 상업적 2차 이용 여부 검토.

## 4. 데이터 모델 (초안)

| 필드 | 타입 | 설명 |
|------|------|------|
| id | UUID / bigint | 내부 PK |
| announced_at | datetime | 고시 시각(원본 `date`) |
| s_pure | int | 살 때 순금(원/3.75g) |
| p_pure | int | 팔 때 순금 |
| p_18k | int | 팔 때 18K |
| p_14k | int | 팔 때 14K |
| raw_json | jsonb | 원본 스냅샷(선택) |
| fetched_at | datetime | 서버 수집 시각 |

---

## 5. 아키텍처 (제안)

- **수집**: Python 스크립트 또는 서버리스(Cloud Scheduler + Cloud Run) — HTTPS POST 후 DB 저장.
- **API**: REST `GET /api/gold-prices?from=&to=&limit=`.
- **프론트**: 정적 호스팅 또는 Next.js — Tabulator·차트 라이브러리.
- **DB**: PostgreSQL / SQLite(소규모).

---

## 6. 일정·마일스톤

1. **M0**: API 연동 POC + 엑셀 100건 내보내기 (현재 저장소 스크립트).
2. **M1**: DB 스키마·일일 배치·간단 웹 목록.
3. **M2**: 기간 검색·차트·에러 알림.

---

## 7. 리스크

- 외부 API URL·스키마 변경 시 수집 깨짐 → 버전 고정·모니터링·fallback 캐시.
- 고시가는 참고용이며 실제 거래가와 다를 수 있음 → **면책 고지** 필수.

---

## 8. 용어

- **3.75g**: 일반적으로 1돈(순금 기준 중량).
- **내가 살 때**: 소비자가 금을 살 때 적용되는 가격(업체 매도가).
- **내가 팔 때**: 소비자가 금을 팔 때 적용되는 가격(업체 매입가).
